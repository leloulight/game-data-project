The Washington State Convention Center is a convention center in Seattle, Washington.

  • Emerald City ComiCon 2008, 2009, 2010, 2011, 2012, 2013, 2014. Will return in 2015.
  • Penny Arcade Expo 2007, 2008, 2009, 2010. 2011, 2012, 2013. Will return in 2014.
  • Sakura-Con 2006, 2007, 2008, 2009, 2010, 2011, 2012, 2013. Will return in 2014.
  • The 41st annual TESOL Convention.
  • WTO Ministerial Conference of 1999
  • NECA Show 2009
  • Phi Theta Kappa Convention 2011
