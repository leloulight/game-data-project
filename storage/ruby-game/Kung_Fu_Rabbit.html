Kung Fu Rabbit is an action video game currently available for the iPod Touch, iPhone, iPad, Android, Windows 7 devices, Nintendo eShop for the Wii U and Nintendo 3DS[1] and PlayStation Vita. It features a Rabbit who is able to practice Kung Fu, and who has to go on a quest and save all the baby rabbits abducted by unknown invaders.

The game has received several accolades from various iOS game critics and received awards.

The game is notable as it is an Indie game awarded by pocket gamers award n°2 in 2012 after angry birds space, and also because it contains references, and is the only game up to this date, to the gameplay of the game Super Meat Boy.[2]
