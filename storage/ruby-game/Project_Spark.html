Project Spark is a "game maker" video game for Microsoft Windows 8.1 and Xbox One. The game was announced during Microsoft's E3 2013 press event, and was launched as a Windows open beta in December 2013, and an Xbox One beta in March 2014.[1] It is based on Kodu Game Lab and was released on October 7, 2014.[2]

Gameplay[edit]

Project Spark Is a digital canvas which can be used to make games, movies and other experiences. A player can download other user-generated content, remix that content or create content of their own. A player can use the Xbox controller, keyboard and mouse, touch-devices, Kinect and SmartGlass to build experiences. Kinect can be used to animate models and record audio. The created environments can contain mountains, rivers, and towns.[3] The player can also create events, like inter-character battles.[3] Created items and objects are able to be shared with other players.[3]

Players choose whether to start from a blank map or a pre-designed level,[4] but always have the tools to customize the topography, add plant and animal life, and program behaviors for specific objects, such as a rock that bounces when a player is nearby.[4] The topography is modified by pushing and pulling the earth, digging through surfaces after changing the view to adjust a wall or create holes in it.[4] Anything added to the terrain reacts to the circumstance, such as grass covering the floor and the vertical surfaces becoming rocks.[4]

Game designer Claude Jerome has said that "the game is all about giving players options", like the ability to add a single flower versus a field of flowers just by resizing the flower paintbrush's size.[4] He added that the game is also about "sharing and playing with the community", and that the difference between Spark and LittleBigPlanet or Minecraft is the core ability to customize the game down to the minutiae of the in-game object actions, which lets the players tell more individual stories.[4] Players can control the game with the Xbox controller, SmartGlass, or a keyboard/mouse combination.[5] The SmartGlass interface mirrors the Xbox One display, but adapts for touch controls when in editor mode.[5]

Worlds created in the game are shareable.[4] Other players who enter can use the world as created, and a duplicate world is created if they would like to make changes.[4]

According to a post from Mike Lescault on the social media website reddit, Project Spark will not require an Xbox Live Gold Membership to be played.[6]

The game will also feature the character Conker the Squirrel.[7]

Development[edit]

Project Spark, developed by Team Dakota,[8] was announced during Microsoft's E3 2013 press event.[3] Registration for its beta began in late June 2013.[9] The game's cross-platform compatibility was demonstrated at Microsoft's Build 2013 developer conference.[10] It is expected for release on Xbox One, Xbox 360, and Windows 8.[10] While there are Windows 8 and Xbox One betas underway, there are no current plans for an Xbox 360 beta. At E3 2014, Team Dakota announced they're hoping for a fall 2014 release. Project Spark became officially available on October 7th 2014, in both PC and Xbox One editions.

Reception[edit]

VentureBeat cited Project Spark as a "good example" of a game that takes advantage of Microsoft's investment in its ecosystem of products.[10] IGN's Mitch Dyer called the game a fascinating "strange mixture of complex and rudimentary".[8]

The video for Linkin Park's "Guilty All the Same" was made using Project Spark and is available to play in game using the community page.
