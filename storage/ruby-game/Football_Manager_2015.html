Football Manager 2015 (abbreviated to FM15) is a football management simulation video game developed by Sports Interactive and published by Sega. It is expected to be released on Microsoft Windows, OS X, Linux - and handheld versions on iOS and Android. The expected release date is November 7, 2014 on Microsoft Windows, Mac and Linux.[1][2] Football Manager Handheld 2015's release date is currently unknown but expected before Christmas 2014.
