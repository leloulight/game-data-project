Colour Cross, released in North America as Color Cross, is a puzzle game for the Nintendo DS. It is a derivative of picross, but requires the user to complete grids using multiple colours with no blank spaces.

The game contains 150 puzzles split between ten categories. The player's time is recorded, with time penalties incurred for mistakes made.

Colour Cross received average reception, with a score of 72/100 on Metacritic.[1]
