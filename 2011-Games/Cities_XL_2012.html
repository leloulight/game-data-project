Cities XL 2012 is a city simulator developed by Focus Home Interactive. Cities XL 2012 is the third game in the Cities XL franchise. The game focuses on a single-player mode, and was released on October 20, 2011.[1] New features include new structures, new maps and a starter guide; additionally, the game has been opened to modding and allows you to share your mods. It will be fully compatible with Cities XL 2011 version with a discounted upgrade available.[2] An expansion pack was released simultaneously.[3]

Development[edit]

The game was officially announced by Focus Home Interactive on July 13, 2011.[2] Cities XL first released on October 8–9, 2009, with the following Cities XL 2011 released on October 14, 2010, then Cities XL 2012 on October 20, 2011. The game was developed by Focus Home Interactive and originally published by Monte Cristo.

Reception[edit]

Cities XL 2012 received mixed reviews from various critics. It currently has a rating of 61/100 on Metacritic.[4]

The biggest weakness of the game is the single core support from the old gaming engine and the poor memory management that slowed the game down in large cities.[citation needed] The fan base has demanded a recompilation of the source code for years.[citation needed]
